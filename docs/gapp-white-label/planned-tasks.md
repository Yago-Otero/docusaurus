# Planned Tasks

## Add a Planned Task (I)

To add a planned task you need to click on the *Properties* button for the new task.

1. Planned task description.
2. Properties button.

## Add a Planned Task (II)

In the *Properties* side panel, click the *Plan this task* button.

1. *Plan this task* button.

## Add a Planned Task (III)

Users must select a day from the calendar to perform the task.

1. Calendar.

## Add a Planned Task (IV)

They must also specify when they will begin the task.

If any ticket requesters have a different time zone set in GLPI, the time will be displayed in relation to the time zone of the user who created the task.

1. Start time for the user creating the task.
2. Start time for the user with another time zone set in GLPI.

## Add a Planned Task (V)

An estimated duration of the task must also be indicated.

1. Duration selector.

## Add a Planned Task (VI)

Finally, a summary of the new task to be created is displayed and the possibility of adding reminders is offered in the time interval that the user selects.

1. Summary view.

## Add a Planned Task (VII)

After creating the task, its configuration is indicated in the *Properties* panel.

1. Planned task configuration.

## Add a Planned Task (VIII)

When a task is planned, the date on which it will be carried out and its duration are also indicated in the main view.

1. Start and duration of the planned task.

## Add a Planned Task (IX)

When saving the scheduled task, a screen is displayed that allows you to add it to the calendar (Google Calendar) of the email associated with the user.

1. Add to calendar button.
2. Skip button.

## Add a Planned Task (X)

Google Calendar automatically collects the data (ticket ID and title, date and planned duration for the task) and allows you to invite more people to the meeting. You can also change the title of the meeting.

1. Ticket ID and title.
2. Date and duration of the meeting.
3. Guest people.
4. Save button.

## Add a Planned Task (XI)

When you save the task, it is displayed at the date and time indicated on the user's calendar.

1. Task scheduled from Gapp and synchronized with Google Calendar.
