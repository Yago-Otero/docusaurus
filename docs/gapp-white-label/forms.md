# Forms: Creating Custom Tickets

## Forms Section

The Forms section displays forms previously created from the GLPI server using the FormCreator plugin. A form will be displayed to all users if it has been set to public, but different types of access can be configured for each form to restrict access by user, group, or profile.

1. Forms section.

## Forms: Attach Files to Tickets

Each form will have different questions. File-type questions can also attach images from different sources:

1. Upload area.
2. You can attach files from different sources: images taken with the camera or existing files (new image, gallery, Google Drive…).
   - Open camera.
   - Local files or other sources.

The first button will open the camera to take the image and attach it.

Multiple files can be selected at once or one at a time. Once the first file is selected, the same screen appears in case the user needs to select more files.

1. Open camera.
2. Local files or other sources.
3. Files already selected.

After selecting the files to attach, the entire form is displayed again so the user can generate the new ticket.

1. Files to attach.
