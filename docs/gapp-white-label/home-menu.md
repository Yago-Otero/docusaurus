# Home & Main Menu

## Home

![Home screen](/img/gapp-white-label/home.jpg)
![Home screen 2](/img/gapp-white-label/home-2.jpg)

This is the home screen in Gapp. Let's explore the features integrated into this interface:

1. **Menu** with access to more features of the application.
2. Search and **access a ticket by ID**.
3. See **New notifications**. *Do not disturb* feature allows you to temporarily silence notifications, ensuring uninterrupted focus when needed.
4. **Tickets**. Easily navigate through all tickets assigned to the logged-in user. This section provides a quick overview of your current workload.
5. **Changes.** Easily track all change requests assigned to the logged-in user. This section offers a clear overview of ongoing or upcoming changes you are involved in, helping you stay on top of planned modifications and their progress.
6. **Dashboards.** Display the native graphs displayed by GLPI (not all of them are supported).
7. **TAM.** The TAM plugin controls the attendance, vacations, and leave of your employees.
8. **Assets.** You can quickly access your assets and open issues on them if you need to use the QR reader. This button also shows all the user's assets in a dropdown without the need to scan QR codes.
9. **Tasks**. From the Tasks section, you can sort and view tasks based on their planning status.
10. **Planning** provides users with a clear view of their daily or weekly schedules, allowing for effective time management and task organization.
11. **Forms.** Forms use the Formcreator plugin that allows the creation of custom forms for easy access.
12. **Running timers**. This feature keeps track of the time spent on your ticket tasks.
13. **Asset Handover** allows for deliveries and returns of assets.
14. **Credits**. This feature assigns credits to tasks, responses, and follow-ups. Ideal for managing hour bonuses, credits are subtracted based on the actual time spent on tasks.
15. **KB** displays previously created knowledge bases on the server, with information and procedures for performing different tasks.

## Main Menu

![Left menu](/img/gapp-white-label/left-menu.jpg)

This menu allows access to several of the functions discussed in the previous section, but also others:

1. **Settings** button.
2. **User.**
3. **Current profile**. This will not be enabled for a one-authorization user.
4. **Current entity.** This will not be enabled for a one-authorization user.
5. **Assigned tickets.**
6. Create a **new ticket**.
7. **Assets.**
8. **TAM.**
9. **Running timers.**
10. **Forms.**
11. **Planning.**
12. **Credits.**
13. **Notification log.**
14. **About**. App version, links, and contact. The usual stuff.
15. **Logout** button.

*Note: You will only be able to access these functions if you have the Gapp White Label version and the respective plugins: TAM, Forms, Running Timers and Credits.*
