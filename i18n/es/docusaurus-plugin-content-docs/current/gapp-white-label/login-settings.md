# Iniciar sesión y configuración

## Iniciar sesión

![Pantalla de inicio de sesión](/img/gapp-white-label/login.jpg)

El inicio de sesión para tu aplicación se personaliza según las preferencias del propietario de la aplicación. Se pueden hacer disponibles varios métodos de autenticación, dependiendo de sus requisitos.

1. **Usuario**. Este campo solicita a los usuarios que ingresen el nombre de usuario único de su cuenta o su dirección de correo electrónico.
2. **Contraseña**. El campo de contraseña es donde los usuarios ingresan su contraseña confidencial.
3. **Botón de enviar**. Los usuarios deben presionar este botón después de ingresar su nombre de usuario y contraseña.
4. **Configuración**. Esta opción permite a los usuarios modificar su cuenta y configuraciones de aplicación.
5. **Iniciar sesión con Google**.
6. **Iniciar sesión con Microsoft 365**.

*Nota: Iniciar sesión con AppleID es compatible a pedido y solo en dispositivos Apple.*

## Configuración (No autenticado)

![Configuración no autenticado](/img/gapp-white-label/settings-not-logged.png)

En la sección de Configuración de nuestra aplicación, se muestra un conjunto de opciones fundamentales para personalizar la experiencia del usuario.

1. **Idioma.** Aquí puedes cambiar el idioma en el que se muestra la aplicación.
2. **ID de instalación.** El ID de instalación es un identificador único para la instancia de la aplicación instalada en el dispositivo de un usuario. Puede ser útil para fines de solución de problemas o soporte.
3. **Versión de Gapp.** Puedes ver más información haciendo clic en él.

## Configuración (Autenticado)

![Configuración autenticado](/img/gapp-white-label/settings-logged.jpg)

Si el usuario está autenticado, se mostrarán más opciones en la sección **Configuración**.

1. **Idioma.**
2. **No molestar**. Al habilitar esta función, se suprimirán las notificaciones de la aplicación, asegurando un enfoque ininterrumpido durante actividades específicas.
3. **Mantenimiento del registro de notificaciones.** Este nuevo botón te permite eliminar el historial de notificaciones de vez en cuando. Cuando lo presiones, se mostrará una ventana emergente en la que deberás indicar el intervalo.
4. **Limpiar caché.** Esta función renueva la sesión del usuario. Es útil para aplicar cambios sin tener que iniciar sesión nuevamente en caso de cambiar perfiles o entidades.
5. **ID de instalación.**
6. **Versión de Gapp.**

*Nota: Regresa a la pantalla anterior haciendo clic en la flecha en la parte superior izquierda.*
