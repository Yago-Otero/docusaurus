# Formularios: Creación de tickets personalizados

## Sección de Formularios

![Sección de formularios](/img/gapp-white-label/forms-section.jpg)

La sección de Formularios muestra formularios creados previamente desde el servidor GLPI usando el plugin FormCreator. Un formulario se mostrará a todos los usuarios si se ha configurado como público, pero se pueden configurar diferentes tipos de acceso para restringir el acceso por usuario, grupo o perfil.

## Formularios: Adjuntar archivos a tickets

![Adjuntar archivos 1](/img/gapp-white-label/forms-attach-1.jpg)
![Adjuntar archivos 2](/img/gapp-white-label/forms-attach-2.jpg)
![Adjuntar archivos 3](/img/gapp-white-label/forms-attach-3.png)
![Adjuntar archivos 4](/img/gapp-white-label/forms-attach-4.jpg)
![Adjuntar archivos 5](/img/gapp-white-label/forms-attach-5.jpg)

Cada formulario tendrá diferentes preguntas. Las preguntas de tipo archivo también pueden adjuntar imágenes de diferentes fuentes:

1. Área de subida.
2. Puedes adjuntar archivos de diferentes fuentes: imágenes tomadas con la cámara o archivos existentes (nueva imagen, galería, Google Drive…).
   - Abrir cámara.
   - Archivos locales u otras fuentes.

El primer botón abrirá la cámara para tomar la imagen y adjuntarla.

Se pueden seleccionar varios archivos a la vez o uno a la vez. Una vez seleccionado el primer archivo, aparece la misma pantalla en caso de que el usuario necesite seleccionar más archivos.

1. Abrir cámara.
2. Archivos locales u otras fuentes.
3. Archivos ya seleccionados.

Después de seleccionar los archivos a adjuntar, se muestra el formulario completo nuevamente para que el usuario pueda generar el nuevo ticket.

1. Archivos a adjuntar.
