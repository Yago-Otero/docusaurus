# AGENTS.md

Docusaurus 3 documentation site (React 19, MDX). Scaffolded from `create-docusaurus`.

## Commands

Use `yarn` (lockfile is `yarn.lock`; npm works too but may drift).

| Command | Description |
|---|---|
| `yarn start` | Dev server with HMR on port 3000 |
| `yarn dev` | Alias for `yarn start` |
| `yarn build` | Static build to `build/` |
| `yarn serve` | Serve built site locally |
| `yarn clear` | Delete `.docusaurus/` and `build/` caches |
| `yarn swizzle` | Eject Docusaurus theme components |
| `yarn deploy` | Build + push to `gh-pages` branch |

No test/lint/typecheck scripts are configured (plain template). `config` uses `@ts-check` JSDoc for type safety.

## Quirks

- `onBrokenLinks: 'throw'` — **build fails** on any broken internal link. Run `yarn build` (not just `yarn start`) to catch dead links before deploy.
- Generated dirs (`.docusaurus/`, `build/`, `.cache-loader`) are gitignored.
- Node >=20 required.
- Has `@docusaurus/faster` 3.10.1 enabled for incremental builds.
- Future flag `v4: true` set for Docusaurus v4 compatibility.

## Structure

| Path | Purpose |
|---|---|
| `docusaurus.config.js` | Site config (title, nav, plugins, theme) |
| `sidebars.js` | Doc sidebar auto-generated from `docs/` |
| `docs/` | MDX documentation pages |
| `blog/` | Blog posts (MDX + `authors.yml`) |
| `src/pages/` | React pages (homepage is `index.js`) |
| `src/components/` | Shared React components |
| `src/css/custom.css` | Theme overrides |
| `static/img/` | Images, favicon, social card |

Custom docs live under `docs/` (gapp-white-label, yago-otero, plus tutorial). The sidebar is `autogenerated` from the `docs/` folder tree.
